JouleHeating

This class computes volumetric heat source due to Joule heating from electric displacement.

Overview

The underlying potential for Joule heating can be written as where is the electric conductivity, is the electric potential, and is the temperature.

At equilibrium, the corresponding heat source is given as which is analogous to the macroscopic heating associated with current flow through a resistor.

Example Input File Syntax

[joule_heating]
  type = JouleHeating
  joule_heating = q
  electric_potential = Phi
  electric_conductivity = sigma
