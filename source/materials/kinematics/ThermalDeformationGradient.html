ThermalDeformationGradient

This class computes the thermal deformation gradient based on the instantaneous CTE.

Overview

The thermal deformation gradient is defined in terms of the instantaneous coefficient of thermal expansion: where is the instantaneous coefficient of thermal expansion. The above equation is numerically integrated in time using the mid point rule.

Example Input File Syntax

[thermal_expansion]
  type = ThermalDeformationGradient
  CTE = '1e-4'
  reference_temperature = 300
  temperature = T
  thermal_deformation_gradient = thermal_deformation_gradient
