MassSource

This class computes the mass source associated with given energy densities for a given species.

Overview

The mass source is defined as where is the reference concentration density, is the total internal energy density, and is the total dissipation density.

Example Input File Syntax

[mass_source+]
  type = MassSource
  mass_source = mu+
  concentration = c+
  energy_densities = 'psi_c+ psi_c-'
  dissipation_densities = 'psi_c+* psi_c-*'
